Description

This website was created from the fretboardgtr python package that I developped some times ago. It allows to create any guitar scales for any tunings from 4 strings to 8 strings instruments very simply. You can display either the notes or the intervals and change the colors. The purpose of this website is to help people to have a better understanding of music but also to compose songs and improvise more easily. When it comes to compose and understand the music we're listening to, knowing where the notes are in the neck and where the intervals are can really make a difference.

Story

As a guitarist and a bassist, I used to play music with friends, analyse song, compose, record and so on. I also used to listen to different genres, from "Math Rock" (Polyphia, Ichika Nito), to more Pop Music (John Mayer, RHCP) but also Metal (BMTH, Crossfaith) to mention just a few. The more i played and listened to music the more i discovered new sounds, new scales or new tunings. So I searched for a way to improvise, find chords positions and new songs more easily. There are plenty of ressources on the Internet for scales and chords but none of them were entirely customizable. I learn the basic of programming at school but i wanted to go deeper, so i decided to create a python package with which we can create the scale we want and decided to also create a website to share at who wants this package.
